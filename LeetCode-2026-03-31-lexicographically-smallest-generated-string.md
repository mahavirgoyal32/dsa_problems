### 📝 Problem Description

You are given two strings, str1 and str2, of lengths n and m, respectively.

A string word of length n + m - 1 is defined to be generated by str1 and str2 if it satisfies the following conditions for each index 0 <= i <= n - 1:


	If str1[i] == &#39;T&#39;, the substring of word with size m starting at index i is equal to str2, i.e., word[i..(i + m - 1)] == str2.
	If str1[i] == &#39;F&#39;, the substring of word with size m starting at index i is not equal to str2, i.e., word[i..(i + m - 1)] != str2.


Return the lexicographically smallest possible string that can be generated by str1 and str2. If no string can be generated, return an empty string &quot;&quot;.

 
Example 1:


Input: str1 = &quot;TFTF&quot;, str2 = &quot;ab&quot;

Output: &quot;ababa&quot;

Explanation:

The table below represents the string &quot;ababa&quot;


	
		
			Index
			T/F
			Substring of length m
		
		
			0
			&#39;T&#39;
			&quot;ab&quot;
		
		
			1
			&#39;F&#39;
			&quot;ba&quot;
		
		
			2
			&#39;T&#39;
			&quot;ab&quot;
		
		
			3
			&#39;F&#39;
			&quot;ba&quot;
		
	


The strings &quot;ababa&quot; and &quot;ababb&quot; can be generated by str1 and str2.

Return &quot;ababa&quot; since it is the lexicographically smaller string.


Example 2:


Input: str1 = &quot;TFTF&quot;, str2 = &quot;abc&quot;

Output: &quot;&quot;

Explanation:

No string that satisfies the conditions can be generated.


Example 3:


Input: str1 = &quot;F&quot;, str2 = &quot;d&quot;

Output: &quot;a&quot;


 
Constraints:


	1 <= n == str1.length <= 104
	1 <= m == str2.length <= 500
	str1 consists only of &#39;T&#39; or &#39;F&#39;.
	str2 consists only of lowercase English characters.

### Problem Understanding:

You are given an integer `n`, and you need to construct a string of length `n` composed of the characters `a`, `b`, and `c`. The character at each position `i` is determined by the following rules:

2. For any index `i` (from 1 to `n-1`), if the character at position `i-1` is:
   - 'a', choose the next character from { 'b', 'c' } such that the result is lexicographically smallest.
   - 'b', choose either 'a' or 'c'.
   - 'c', choose either 'a' or 'b'.

The goal is to generate the lexicographically smallest string of length `n`.

The optimal way to solve this problem is to use an iterative approach to build the string character by character, applying the rules provided at each step.

#include <iostream>
#include <string>
using namespace std;

string generateString(int n) {
    if (n == 0) return ""; // Edge case for n = 0
    
    string result(n, ' '); // Create a string of size n
    result[0] = 'a'; // First character is always 'a'
    
    for (int i = 1; i < n; ++i) {
        if (result[i - 1] == 'a') {
            result[i] = 'b'; // If previous is 'a', choose 'b'
        } else if (result[i - 1] == 'b') {
            result[i] = 'a'; // If previous is 'b', choose 'a' (to keep it lexicographically smaller)
        } else {
            result[i] = 'a'; // If previous is 'c', choose 'a'.
        }
    }
    
    return result; // Return the generated string
}

    return 0;
